# Game Circles技术服务支持

Game Circles是一款极其简单易于操作的休闲益智类小游戏。 你只需要简单地触摸母球的操作即可放大圆圈，然后吸收相同颜色的球，轻触球外部分缩小母球以避免吸收了不同颜色的球而导致游戏终止。紧张又刺激的背景音乐，易于理解但又有挑战性的操作，快来试试你能够得到多少分，快来享受轻松愉快的时光吧！
